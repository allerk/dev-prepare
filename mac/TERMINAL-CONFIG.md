## Zsh — `.zshrc`

### Framework

- **Oh My Zsh** installed at `$HOME/.oh-my-zsh`
- **Theme**: `powerlevel10k/powerlevel10k`

### Plugins

```zsh
plugins=(git)
```

### Instant Prompt

Powerlevel10k instant prompt is enabled. It sources the cached prompt at startup for near-instant shell load:

```zsh
if [[ -r "${XDG_CACHE_HOME:-$HOME/.cache}/p10k-instant-prompt-${(%):-%n}.zsh" ]]; then
  source "${XDG_CACHE_HOME:-$HOME/.cache}/p10k-instant-prompt-${(%):-%n}.zsh"
fi
```

### p10k Config Source

At the end of `.zshrc`, the Powerlevel10k config is loaded:

```zsh
[[ ! -f ~/.p10k.zsh ]] || source ~/.p10k.zsh
```

### Full `.zshrc`

```zsh
# Enable Powerlevel10k instant prompt. Should stay close to the top of ~/.zshrc.
if [[ -r "${XDG_CACHE_HOME:-$HOME/.cache}/p10k-instant-prompt-${(%):-%n}.zsh" ]]; then
  source "${XDG_CACHE_HOME:-$HOME/.cache}/p10k-instant-prompt-${(%):-%n}.zsh"
fi

export ZSH="$HOME/.oh-my-zsh"

ZSH_THEME="powerlevel10k/powerlevel10k"

plugins=(git)

source $ZSH/oh-my-zsh.sh

# To customize prompt, run `p10k configure` or edit ~/.p10k.zsh.
[[ ! -f ~/.p10k.zsh ]] || source ~/.p10k.zsh
```

## Installation Summary

### Dependencies

| Tool | Install |
|---|---|
| Zsh | pre-installed on macOS |
| Oh My Zsh | `sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"` |
| Powerlevel10k | `git clone --depth=1 https://github.com/romkatv/powerlevel10k.git ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/powerlevel10k` |
| Nerd Font | Download from [nerdfonts.com](https://www.nerdfonts.com/) |
| Tmux | `brew install tmux` |
| Catppuccin tmux | `git clone https://github.com/catppuccin/tmux ~/.config/tmux/plugins/catppuccin/tmux` |

### Quick Setup

```bash
# 1. Install Oh My Zsh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

# 2. Install Powerlevel10k
git clone --depth=1 https://github.com/romkatv/powerlevel10k.git \
  ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/powerlevel10k

# 3. Copy .zshrc and .p10k.zsh to ~

# 4. Install tmux (macOS)
brew install tmux

# 5. Install Catppuccin tmux theme
mkdir -p ~/.config/tmux/plugins
git clone https://github.com/catppuccin/tmux \
  ~/.config/tmux/plugins/catppuccin/tmux

# 6. Copy tmux.conf to ~/.config/tmux/tmux.conf

# 7. Reload shell
source ~/.zshrc
```
